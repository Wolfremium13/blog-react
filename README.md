# 🔗 [wolfremium.dev](https://www.wolfremium.dev/)

## 🖥 Setup

Node version: v18.2.0

Development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the
result.

## 📦 Minify images

```bash
npm run gulp
```
